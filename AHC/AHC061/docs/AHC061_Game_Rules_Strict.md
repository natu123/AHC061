# AHC061 Game Rules Strict

## 目的
- AHC061 のゲームルールを、ローカルテスタ実装に沿って厳密に整理します。
- 実装上の正本は `N52XwIfp_local/tools/src/lib.rs` とします。

## 固定制約
- `N = 10`（盤面 `10x10`）
- `M = 2..8`（プレイヤー数）
- `T = 100`（ターン数）
- `U = 1..5`（レベル上限）
- `sum(V[i][j]) = 100000`
- 初期状態で各プレイヤーは異なる1マスを所有し、初期レベルは1

## 状態定義
- `pos[p]`: プレイヤー `p` の現在位置
- `owner[x][y]`: マス `(x, y)` の所有者（`-1` は未領土）
- `level[x][y]`: マス `(x, y)` のレベル
- `selected[p]`: そのターンにプレイヤー `p` が選んだ移動先

## 移動可能先（`is_valid_move` / `get_candidates`）
- 移動先は盤内である必要があります。
- 他プレイヤーが現在いるマスは移動先にできません。
- 候補列挙は BFS で行います。
- BFS は `pos[p]` から開始します。
- BFS の展開は「現在マスの `owner == p`」のときだけ 4 近傍へ進みます。
- 4 近傍の順序は `(0,1) -> (1,0) -> (0,-1) -> (-1,0)` です。
- 到達済みマスのうち、他プレイヤー現在位置でないものを候補に含めます。

## 1ターン更新（`update_state`）
1. 全プレイヤーの移動先について、マスごとの人数を集計します。
2. 同一マスに2人以上集まった場合、現所有者以外のプレイヤーを回収フラグにします。
3. 回収されていないプレイヤーだけ、選択マスに対して占領・強化・攻撃を実行します。
4. 攻撃失敗（レベルを0にできない）したプレイヤーは回収フラグになります。
5. 回収フラグのプレイヤーは元の位置へ戻ります。
6. `pos` をターン後位置に更新します。

## マス作用の詳細
- 未領土（`owner == -1`）:
  - `owner = p`
  - `level = 1`
- 自領土（`owner == p`）:
  - `level < U` のとき `level += 1`
  - `level == U` のとき変化なし
- 他領土（`owner != p`）:
  - `level -= 1`
  - `level == 0` になったら奪取（`owner = p`, `level = 1`）
  - `level > 0` のままなら攻撃側は回収（元位置へ戻る）

## 競合時の扱い
- 同一移動先に複数プレイヤーがいる場合、現所有者だけが残ります。
- 非所有者はそのターン回収されます。
- 競合解決後にマス作用を適用します。

## 出力フォーマット（player0）
- 1ターンにつき `x y` を1行出力します。
- 合計 `T=100` 行を想定します。
- 行頭 `#` はコメントとして無視されます。
- 1行に余分なトークンがあるとエラーです。

## エラー判定（`tester`）
- 範囲外移動: `Player X attempted to move to out-of-bounds position (...)`
- 不正移動: `Player X attempted invalid move from (...) to (...)`
- 余分トークン: `Too many tokens in line: ...`
- 異常終了: `Your program has terminated unexpectedly`
- エラー時のスコアは `Score = 0`

## スコア定義
- 最終盤面で各プレイヤーのスコアを `sum(V[i][j] * level[i][j])` で計算します。
- `S0` を player0 のスコア、`SA` を AI（player1..M-1）の最大スコアとします。
- 絶対スコア:
  - `round(1e5 * log2(1 + S0 / SA))`

## AI（player1..M-1）行動モデル
- 評価値:
  - 未領土: `V * wa`
  - 自領土かつ `level < U`: `V * wb`
  - 自領土かつ `level == U`: `0`
  - 他領土かつ `level == 1`: `V * wc`
  - 他領土かつ `level >= 2`: `V * wd`
- `r1 < eps` ならランダム選択、それ以外は最大評価値から選択（同値ランダム）
- 同値判定は `tolerance = 1e-9 * max(abs(max_score), 1.0)` を使用
