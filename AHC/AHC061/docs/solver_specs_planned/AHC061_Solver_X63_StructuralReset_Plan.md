# AHC061 Solver X63 Structural Reset Plan

## 目的
- `x56~x62` 系の微調整寄り改善での上限到達感に対して、構造を入れ替えた新規戦略で上位帯へ抜本到達を狙う。
- `quick/full` の時間効率を維持しつつ、`+50%` 前提に対し「mean/median/min` の3軸同時改善余地を持つ系統を再構築する。

## 狙い / 別系統性
- `x56~x62` が最適化軸をほぼ共有しているため、次世代系統では次を同時に見直す。
  - 初期解生成（seed依存性の低減と多様化）
  - 遷移選好（局所最適に偏らない新規性）
  - 評価関数（`mean/median/min` の三重最適化）
- 上記3軸の入れ替えにより、既存経路の再利用ではなく異質な改善メカニズムを形成する。

## 期待効果
- `x56` / `x57` のような `mean` 中心寄りの重なり改善から脱却し、`median/min` を悪化させない改善を目指す。
- 同一コスト内で `gain/sec` を上げるため、`quick` 優先フェーズで `top candidate` の比較速度を維持する。
